## Brand Overview

**BitBuilder Cloud** is a distributed, vendor-agnostic, end-to-end encrypted, multi-cloud infrastructure platform built on systemd's proven virtualization and containerization capabilities.

### Brand Personality
- **Technical**: Professional, precise, engineering-focused
- **Secure**: Trust, privacy, zero-knowledge architecture
- **Modern**: Clean, minimal, forward-thinking
- **Open**: Transparent, community-driven, Unix Philosophy

### Visual Identity Principles
- **Minimalism**: Clean lines, no unnecessary decoration
- **Technical Precision**: Terminal/code aesthetic
- **Modularity**: Components that can be combined
- **Adaptability**: Works across digital and print media

## Typography

### Primary Typeface: System UI Stack

For optimal readability and performance across platforms:

```css
font-family: system-ui, -apple-system, BlinkMacSystemFont,
             'Segoe UI', Roboto, 'Helvetica Neue', Arial,
             sans-serif;
```

### Monospace (Code/Terminal):

```css
font-family: 'SF Mono', Monaco, 'Cascadia Code',
             'Roboto Mono', Consolas, 'Courier New',
             monospace;
```

### Font Weights

- **Light (300)**: Captions, secondary text
- **Regular (400)**: Body text, paragraphs
- **Medium (500)**: Subheadings, emphasis
- **Semibold (600)**: Headings, CTAs
- **Bold (700)**: Major headings, high emphasis

### Type Scale

- **Display**: 48px / 3rem (line-height: 1.2)
- **H1**: 36px / 2.25rem (line-height: 1.3)
- **H2**: 30px / 1.875rem (line-height: 1.3)
- **H3**: 24px / 1.5rem (line-height: 1.4)
- **H4**: 20px / 1.25rem (line-height: 1.4)
- **Body**: 16px / 1rem (line-height: 1.6)
- **Small**: 14px / 0.875rem (line-height: 1.5)
- **Caption**: 12px / 0.75rem (line-height: 1.4)

## Logo Usage

### Minimum Sizes

- **Digital**: 120px width minimum
- **Print**: 1 inch (25.4mm) width minimum
- **Favicon**: 32×32px minimum (use square version)

### Preferred Placements

1. **Top Left**: Primary position for headers and navigation
2. **Center**: Hero sections, splash screens
3. **Bottom Right**: Footer, documentation pages

### Background Requirements

#### For Black Logos:
- Use on white or light backgrounds
- Minimum contrast ratio: 7:1
- Avoid busy or patterned backgrounds

#### For White Logos:
- Use on dark backgrounds (#252A34 or darker)
- Minimum contrast ratio: 7:1
- Ensure sufficient contrast

### Clear Space

Maintain clear space around the logo equal to the height of the terminal icon (the square box in the logo). This ensures the logo has room to breathe and maintains visual impact.

```

```

## Do's and Don'ts

### ✅ DO

- Use the provided logo files without modification
- Maintain proper aspect ratios when scaling
- Ensure adequate contrast with backgrounds
- Use appropriate logo variation for context
- Keep clear space around logo
- Use vector (SVG) format whenever possible
- Request approval for custom applications

### ❌ DON'T

- Stretch, distort, or skew the logo
- Change colors or add effects
- Rotate the logo
- Add drop shadows, glows, or gradients
- Place logo on busy backgrounds
- Use low-resolution pixelated versions
- Outline or add borders
- Rearrange logo elements
- Use outdated logo versions

### Common Mistakes

#### ❌ Wrong: Stretched Logo
The logo should maintain its aspect ratio.

#### ❌ Wrong: Modified Colors
Do not change logo colors to match your brand.

#### ❌ Wrong: Added Effects
No drop shadows, 3D effects, or gradients.

#### ❌ Wrong: Low Contrast
Ensure sufficient contrast with background.

## File Formats & Specifications

### Available Formats

#### SVG (Scalable Vector Graphics)
- **Use**: Web, digital, any scalable application
- **Benefits**: Infinite scalability, small file size, crisp at any resolution
- **Locations**:
  - `/exports/optimized-svg/` - Clean, optimized versions
  - `/exports/square-svg/` - Square aspect ratio
  - `/exports/circular-svg/` - Circular cropped
  - `/exports/inverted-svg/` - Color-inverted versions

#### PNG (Portable Network Graphics)
- **Use**: Digital applications requiring raster images
- **Available Sizes**: 16×16, 32×32, 64×64, 128×128, 256×256, 512×512, 1024×1024, 2048×2048
- **Background**: Transparent (recommended) or solid color
- **Generate**: Use `svg-to-png-converter.html` tool

#### ICO (Icon Format)
- **Use**: Favicons, Windows icons
- **Location**: `/exports/favicons/`
- **Sizes**: Multi-resolution (16, 32, 48, 256)

### Export Settings

#### For Web:
- **Format**: SVG (preferred) or PNG
- **PNG Resolution**: 2× actual display size for Retina/HiDPI
- **Optimization**: Use SVGO for SVG, TinyPNG for PNG

#### For Print:
- **Format**: SVG (for infinite scaling) or high-res PNG (300 DPI minimum)
- **Color Mode**: CMYK for offset printing, RGB for digital printing
- **Bleed**: Add 3mm bleed for cut materials

#### For Social Media:
- **Profile Pictures**: 512×512px PNG (circular or square)
- **Banners**: Platform-specific dimensions
- **Thumbnails**: 256×256px PNG minimum

## Applications

### Digital Applications

#### Websites
- **Header**: Use horizontal variation (01 or 02)
- **Footer**: Use compact variation (09, 10, or 11)
- **Favicon**: Use square version at 32×32px minimum

#### Email Signatures
- Use compact horizontal version
- Maximum width: 200px
- Link to website
- See `/templates/email-signatures/` for HTML templates

#### Social Media
- **Profile Picture**: Square or circular version, 512×512px
- **Cover Photo**: Full brand mark with ample padding
- **Posts**: Use brand-appropriate variation with tagline

### Print Applications

#### Business Cards
- Use full brand mark with tagline
- See `/templates/business-cards/` for templates
- Standard sizes: US (3.5"×2"), EU (85×55mm)

#### Letterhead
- Top left or center placement
- See `/templates/letterhead/` for templates
- A4 and Letter sizes available

#### Marketing Materials
- Brochures: Full brand mark on cover
- Flyers: Large variation for impact
- Banners: High-resolution PNG or vector

### Technical Documentation

- Use horizontal variation with terminal icon
- Monochrome (black or white) preferred
- Combine with monospace typography
- See compact variations for badges
